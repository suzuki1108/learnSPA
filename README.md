# アプリケーションの概要
楽天BooksAPIを利用した、本棚管理用のWebアプリケーションです。
※SpringBoot学習用に作成したアプリケーションをAPI化し、SPA化しました。

# 使用技術
## フロント
React.js
node.js
Material-UI
## バックエンド
Java8
SpringBoot
PostgreSQL
## サーバ
heroku
